This index tracks the source documents that informed the architecture, domain definitions, and system design for Reach Developments Station. These documents are **references** — they are not runtime dependencies and should not be committed to the repository root.

Key findings from these documents are summarized in [`key-findings-summary.md`](key-findings-summary.md).
